# حصنك اليومي ✨ - PWA

## خطوات الرفع على Netlify (مجاناً)

### 1. افتح netlify.com وسجّل دخول
### 2. اضغط "Add new site" → "Deploy manually"
### 3. اسحب فولدر hesnk-pwa كله وحطه في المربع
### 4. انتظر ثواني وهيديك رابط زي: https://amazing-name-123.netlify.app
### 5. افتح الرابط من موبايلك

---

## تنزيل التطبيق على الموبايل

### أندرويد (Chrome):
- افتح الرابط
- هيطلعلك بانر "نزّل التطبيق" تلقائي
- أو: القائمة (⋮) → "Add to Home screen"

### iPhone (Safari):
- افتح الرابط في Safari (مش Chrome)
- اضغط زر المشاركة (□↑)
- اختر "Add to Home Screen"
- اضغط "Add"
